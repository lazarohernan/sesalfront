🔧 Uso con JavaScript

// Acceder al widget después de cargar const widget = document.querySelector('bi-sesal-completo'); // Actualizar configuración dinámicamente widget.updateConfig({ anio: 2023, departamento: 'Atlántida', theme: 'dark' }); // Escuchar eventos del widget widget.addEventListener('config-changed', (event) => { console.log('Configuración actualizada:', event.detail); });

📋 Notas Importantes

⚠️ Servidor Backend: Asegúrate de que el backend esté corriendo en el puerto 4000 y permita CORS desde el dominio donde incrustes el widget.
🔗 URLs: Actualiza las URLs del backend en el widget si es necesario. Actualmente apunta a http://localhost:4000.
🎨 Personalización: Los estilos del widget están encapsulados en Shadow DOM, por lo que no interfieren con el sitio padre.
